# Switch 开关

## 代码展示

```javascript
<alice-switch v-model="value">开关</alice-switch>
```



## Switch API

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|------|
| value | 开关选中状态 | `Boolean` | `false` | 
| disabled |  是否为禁用状态  | `Boolean` | `false` |

## Switch Event

| 事件名 | 说明 | 参数 |
|------|------|------|
| input | 当绑定值变化时触发的事件 | 当前组件的值|
| on-change | 当绑定值变化时触发的事件 | 当前组件的值|
